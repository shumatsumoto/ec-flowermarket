## :pencil2: Overview

I created an EC site. Pay.jp has a credit card payment function.

## :hammer: Requirement

- macOS
- Rails 6.0.3.4
- [Material Design for Bootstrap4](https://mdbootstrap.com/)

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/ec-flowermarket.git
```
Move to directory
```
$ cd ec-flowermarket
```
Install the library
```
$ bundle install
```
Launch a local server
```
$ rails s
```
Access URL： 
http://localhost:3000

## :railway_car: Features

- Article posting
- Credit card payment
- Login / Logout
- SignUp
